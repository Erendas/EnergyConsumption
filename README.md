# Explanation
   Energy Consumption Prediction Project
 An energy consumption prediction project aims to forecast how much energy will be used in the future. The forecast can be weekly, monthly or yearly.
 The main goal is to help organizations reduce costs, improve efficiency, and support sustainability by making smarter energy decisions.

# Project Objective
  This project usually solves one or more of these business problems:

 Predict weekly/monthly/yearly energy demand
 Detect unusual consumption patterns (possible equipment faults or waste)
 Improve peak-load management (avoid expensive demand spikes)
 Optimize HVAC, lighting, and production schedules
 Support renewable integration (solar/wind balancing)
 
## Installation
 1) Clone the repository
  git clone https://github.com/your-username/novashop-frontend.git
 2) Go into the project folder
  cd novashop-frontend
 3) Install dependencies
  npm install
 4) Start development server
  npm run dev
